# math-game
Math revision game targeted at Key Stage 2 students looking to help them for the 11 plus entrance exams

Three main sections containing:

  Basic Arithmetic
  
    Addition
    
    Subtraction
    
    Multiplication
    
    BIDMAS
    
  Fractions
  
    Adding Fractions
    
    Subtracting Fractions
    
    Multiplying Fractions
    
    Comparing Fractions
    
  Problem Solving
  
    Money Problems
    
    Weights
    
    Recipes
